# Issue #p1-008: 仪表盘 UI — 小说管理 + 评估历史页

## What to build

仪表盘前端页面：小说列表/详情管理 + 评估历史分页浏览。

1. `frontend/src/app/(dashboard)/novels/page.tsx` — 小说列表页：
   - 卡片网格展示用户的所有小说
   - 每张卡片：书名、题材标签、章节数、最近评估时间
   - "创建新小说"按钮 → 弹窗输入书名 + 题材下拉选择
   - 空状态：引导文字"还没有小说，开始第一次评估吧"
   - 点击卡片进入小说详情
2. `frontend/src/app/(dashboard)/novels/[id]/page.tsx` — 小说详情页：
   - 小说标题 + 题材
   - 章节列表（按 number 排序），每行：章节标题、评估次数、最近评估分数
   - 点击章节展开该章节的所有评估报告列表
   - 点击报告进入完整报告页（复用 ReportCard）
3. `frontend/src/app/(dashboard)/history/page.tsx` — 评估历史页：
   - 表格展示：日期、小说、章节、评分、操作（查看报告）
   - 分页控件（上一页/下一页）
   - 按小说下拉筛选
   - 空状态：引导文字
   - 点击行进入完整报告视图
4. 所有页面使用 `(dashboard)/layout.tsx` 的受保护布局
5. 加载态 + 错误态均覆盖
